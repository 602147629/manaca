percentageproperty 
percentage:uint  [read-only]

Returns the execution progress in percent.

If this process has not started execution yet, 0 will be returned.

If this process has already started exeuction and the progress is evaluable, a value between 0 and 100 will be returned, depending on the current execution state.

If this process has already finished execution, 100 will be returned.

addProcess()method 
public function addProcess(process:IProcess):void

Adds the given process to execute to this batch.

It is possible to add the same process more than once. It will be executed as often as you add it.

Note that the given process will not be added if it is this batch itself or if it is null

Parameters

process:IProcess — the process to add
