TimeProcessor provides methods to measure execution time.

The concrete implementation needs to take care of startTime, endTime, percentage.

Property Detail
durationproperty
duration:uint  [read-only]

Returns the time needed for the execution (until now).

If execution has already finished, the totally needed execution time will be returned.

Otherwise the time needed from the start of the execution until this point in the execution will be returned.
