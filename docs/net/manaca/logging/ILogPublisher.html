Interfacepublic interface ILogPublisher
Implementors AbstractLogPublisher, AlconPublisher, LongingPublisher, Output, TracePublisher

The ILogPublisher interface is used to actually log messages. you need different handlers for different output targets. Output targets can be everything, any type of file, a database, a custom, trace, output console and so on.
